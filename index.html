Capstone Project

Visitor Collection Tool

The Visitor Collection Tool was developed by Myself, Ian Shepard, and Kylie Harmon in 2020 for Innovation Connector in Muncie, IN


The purpose of this software is to collect attendance information for events hosted by Innovation Connector, so that they can better understand who attends these events, which events are more popular than others, and to help determine why some attendees might quit certain series' of classes that are hosted


The software includes an event check in and registration system for attendees. It also includes a dashboard for staff members to manage the creation of events, registering attendees, and to view analytics of collected data. Everything was designed with simplicity in mind, as a wide range of people will be using the software.


This software was developed as a web application using PHP as the server-side language and MySQL for the DBMS
